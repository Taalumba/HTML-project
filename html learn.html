Selectors, properties and values

Selectors are to CSS, what tags are to HTML. And each selector has properties found inside {..:..} these. And properties have values after the colon, while semi-colons seperate values.

Colors

CSS brings 16 million plus colours to ya... And these can be specified by name, RGB value or hex code. Eg, all the following will give the same red but are expressed differently:

  • red
  • rgd(255,0,0)
  • rgb(100%,0%,0%)
  • #ff0000
  • #f00

Some colours have predefined names, but these are just the major solid ones and are limiting. getting all the colours requires use of rgb values or the hexadecimal values. CSS3 and up also allow you to control hue, saturation and lightness (HSL).

Colour and background colour

Colours can be added to both text and background; unfortunately, we must spell it "color" like Americans, and not "colour" like normal people. *sigh*. We shall now make all major headings yellow with a blue background using the CSS file, and then modify.

Text

Text size and shape can also be changed. You change the font family; like with headings 2 here. Another thing you can do is modify font size; duh. Warning, dont make headings using the < p > tag. use the < h1,2,3 > tags. Its not recommended to just use p.

Thirdly, we can change font-weight, which tells us if font is bold or nah; there are actually several values of these. Values between bolder and lighter.

Bold

And the heading "Bold", here has been modifed for weight and style. we can also use the "{font-variant: small caps}" and the "{text-transform: upper case}" properties and values. Externally embedded of course. There is also {text-decoration} which determines if text has a line running through, over or under it, with the "underline", "overline", and "line-through" values. You can also specify "none" with this one. But really, underlines should only be used for links.

As already stated, there is a text-transform property, which changes the case of text and has capitalize, uppercase, lowercase and none values.
